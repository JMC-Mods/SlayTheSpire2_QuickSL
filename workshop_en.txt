[h1]Quick SL (Multiplayer / Controller / Configurable Hotkey Support)[/h1]
[h2] 1. Overview [/h2]
This is a quick SL MOD with multiplayer support, configurable hotkeys (including key combinations), and native Steam Input support for controller button binding!

[url=https://github.com/JMC2002/SlayTheSpire2_QuickSL] GitHub Repository [/url]
[url=https://www.bilibili.com/video/BV1BnwXziEsc] Demo Video (Bilibili) [/url]

[h2] 2. Features [/h2]
[list]
    [*] Provides hotkey configuration with an enable checkbox in the JmcModLib MOD settings screen
    [img]https://images.steamusercontent.com/ugc/10453256321932357138/8B20814E4BB31E1B1E1D39E2CD806B7A72D2D3CA/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] The default keyboard hotkey is `F5`; controller button binding requires Steam Input
    [img]https://images.steamusercontent.com/ugc/17878268037868162019/F7FBC6396C8559F6FC090812E6BC6B5C58D32339/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] Provides an `S & L` entry in the in-run pause menu, which can directly trigger quick SL
    [img]https://images.steamusercontent.com/ugc/10394527552730582370/BFC4D65DAA21CB3E7339DDB502CC9C0CE1B900A4/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] Supports synchronized quick SL in multiplayer initiated by either the host or a client; single-player and multiplayer reuse the same hotkey and pause menu entry
    [img]https://images.steamusercontent.com/ugc/17361571361743273558/3C2CBC615D395709A3ADFE83C732ACA63437A895/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] In multiplayer, you can configure whether the host allows clients to initiate quick SL, whether the host is prompted first when a client initiates it, and whether other clients are prompted before SL
[/list]

[h2] 3. Notes [/h2]
[list]
    [*] Because of the game's mechanics, installing any MOD for the first time can cause save desynchronization. You can use [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747533700] BetterSaveSlots [/url] to solve this
    [*] This MOD requires the following dependency:  
• [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747526103] JmcModLib [/url]  
    [*] The localization content of this MOD (including the workshop page and changelog), except Chinese, is generated by ChatGPT. It currently supports all languages officially supported by the game. If you think any wording is inaccurate or would like to contribute your own translation, feel free to open an issue / PR on GitHub or contact me directly
[/list]

[h2] 4. Compatibility [/h2]
[list]
    [*] This mod makes extensive use of reflection and may stop working after game updates. If you find that the mod no longer works, please report it promptly.
[/list]
